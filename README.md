# MK Fraud single platform

One Next.js application serving the MK Fraud public website at the domain root and the Fraud Readiness product under `/score`.

The product retains accountless assessment, deterministic scoring, report workflows and MK admin review. Phase 14 remains disabled, and migration 0017 is not part of consolidation validation.
